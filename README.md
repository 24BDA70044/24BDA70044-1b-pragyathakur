# Responsive Banking UI

## Aim
To develop a responsive banking interface that supports deposit and withdrawal operations using **HTML, CSS, and JavaScript**.

---

## Objective
- Design the user interface using **CSS Flexbox**
- Implement **Deposit and Withdraw** functionality using JavaScript
- Validate user input and display appropriate messages
- Make the layout responsive using **media queries**

---

## Hardware / Software Requirements

### Hardware
- PC or Laptop

### Software
- Visual Studio Code (VS Code)
- Web Browser (Google Chrome)

---

## About the Program
This application displays a bank balance and allows users to **deposit or withdraw money**.  
JavaScript manages balance updates and input validation, while **CSS Flexbox** ensures a clean and responsive layout across different screen sizes.

---

## Files Used
- `index.html` – Defines the structure of the banking interface
- `style.css` – Handles styling and responsiveness
- `script.js` – Implements deposit, withdrawal, and validation logic

---

## Working
- The initial bank balance is displayed on the screen
- The user enters an amount in the input field

### Deposit
- Adds the entered amount to the current balance

### Withdraw
- Subtracts the entered amount if sufficient balance is available
- Displays an error message for invalid input or insufficient funds

---

## Output
A responsive banking interface that displays:
- Current balance
- Input field for amount
- Deposit and withdraw buttons
- Validation and error feedback messages

---

## Conclusion
This project demonstrates the use of **CSS Flexbox for layout**, **DOM manipulation**, and **basic state management** using JavaScript to build an interactive and responsive banking application.
